icmp_receiver.receive_target_packet() → Packet[source]

Sniffs and receives a single target ICMP packet that is sent by icmp_sender.py.

The packet is filtered by the following conditions:

  1. Sent by the sender host to receiver host

  2. It has the TTL value of 1

  3. It is an ICMP echo-request packet

Returns:

The received ICMP target packet.

Return type:

Packet

 4def receive_target_packet() -> Packet:
 5    """Sniffs and receives a single target ICMP packet
 6    that is sent by `icmp_sender.py`.
 7
 8    The packet is filtered by the following conditions:
 9
10    1. Sent by the sender host to receiver host
11    2. It has the TTL value of 1
12    3. It is an ICMP echo-request packet
13
14    Returns:
15        Packet: The received ICMP target packet.
16    """
17    filter = "icmp"
18    filter += " and src host sender"
19    filter += " and dst host receiver"
20    filter += " and ip[8] == 1"
21    filter += " and icmp[icmptype] == icmp-echo"
22    packets = sniff(filter=filter, count=1)
23    return packets[0]
